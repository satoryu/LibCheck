制定日: 2026年6月28日

本利用規約（以下「本規約」）は、satoryu（以下「開発者」）が提供する Web アプリケーション「LibCheck」 （以下「本サービス」）の利用条件を定めるものです。利用者は、本サービスを利用することにより本規約に同意したものとみなされます。

1. 本サービスの内容

本サービスは、書籍の ISBN をもとに、利用者が登録した図書館における蔵書の貸出状況を確認できる 無償のツールです。蔵書・図書館の情報はカーリル株式会社が提供する カーリル図書館 API 等の外部サービスに依拠しており、 本サービスは「現状有姿」で提供されます。

2. アカウント・利用条件

  • 本サービスの利用には Google アカウントでのログインが必要です。
  • 利用者は、自己の責任においてアカウントを管理するものとします。
  • 個人情報の取り扱いは プライバシーポリシー に従います。

本サービスは、Amazon.co.jp を宣伝しリンクすることによって紹介料を獲得できる Amazon アソシエイト・プログラムの参加者です。
