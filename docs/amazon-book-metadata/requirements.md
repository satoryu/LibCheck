# 蔵書検索結果画面への書影・タイトル・購入リンク表示 — Requirements

## Problem Statement

ISBN による蔵書検索結果画面 (`BookSearchResultPage`) には現在、`ISBN: xxxx` の文字列と各登録図書館の蔵書状況しか表示されない。利用者は「いま検索している本がどの本なのか」を視覚・書誌情報から確認できず、ISBN の取り違えに気づきにくい。

書影（カバー画像）・タイトル・購入導線（価格情報の代替）を表示し、検索対象の本を直感的に確認できるようにする。

## データ取得手段に関する重要な制約

- 当初は Amazon Product Advertising API (PA-API 5.0) からタイトル・書影・価格を取得する想定だった。
- しかし **PA-API 5.0 は 2026-05-15 に廃止済み**であり利用できない。後継の Amazon Creators API は OAuth2 認証に加え「直近30日で適格販売10件」という参加要件があり、本アプリでは現実的でない。
- ユーザー合意のもと、**認証不要のハイブリッド方式**を採用する。

| 項目 | データ源 | 方式 |
|---|---|---|
| 書影 | Amazon 画像CDN | ISBN-10 から直URL（`<img src>`、フェッチ不要） |
| タイトル(・著者・出版社) | OpenBD API | ブラウザから直接 fetch（CORS オープン・APIキー不要） |
| 価格 | （表示しない） | 「Amazonで見る」リンクで代替 |
| 購入リンク | Amazon | ISBN-10 から `/dp/` URL を生成（導出不可時は検索 URL） |

## Requirements

### Functional

- FR-1: 結果画面で、検索中の ISBN に対応する書影を表示する。
- FR-2: 結果画面で、書籍タイトル（取得できれば著者も）を表示する。
- FR-3: 結果画面に「Amazonで見る」リンク（ボタン）を表示し、新規タブで Amazon 商品ページを開く。
- FR-4: 書影が取得できない場合でも、レイアウトを崩さずプレースホルダで代替し、タイトル・リンクは可能な範囲で表示する。
- FR-5: タイトル取得（OpenBD）が失敗・該当なしでも、書影と Amazon リンクは ISBN から導出して表示する。

### Non-Functional

- NFR-1: 書影・タイトル取得の失敗が、中核機能である蔵書状況表示を一切妨げないこと（グレースフルデグラデーション）。
- NFR-2: 外部 API キー・シークレットを必要としない（クライアントから OpenBD を直接呼び、画像・リンクは ISBN から導出）。
- NFR-3: Clean Architecture（domain/data/presentation）の依存方向を維持する。
- NFR-4: Vitest による単体・結合テストを備える（TDD）。

## Constraints

- 価格は表示しない（Amazon を正規に取得する手段が現状ないため）。リンクで代替する。
- 新規 Azure Function・staticwebapp 設定・vite proxy・Azure appsettings は変更しない。
- 979 始まりの ISBN-13 は ISBN-10 を持たないため、書影は表示できず、購入リンクは Amazon 検索 URL にフォールバックする。

## Acceptance Criteria

- AC-1: 既知の ISBN（例 `9784873117584`）で結果画面を開くと、書影・タイトルが表示され、「Amazonで見る」が `https://www.amazon.co.jp/dp/4873117585` を新規タブで開く。
- AC-2: OpenBD に該当の無い ISBN でも、画面が崩れず、書影（または代替）とリンクが表示される。
- AC-3: 書影が存在しない ISBN（Amazon が画像なしを返す）でも、プレースホルダ表示となり破綻しない。
- AC-4: メタデータ取得が失敗しても、蔵書状況（貸出可/貸出中等）は従来どおり表示される。
- AC-5: すべての Vitest テストと型チェック（`npx tsc -b`）が通る。

## User Stories

- US-1: 利用者として、検索結果画面で本の表紙とタイトルを見て、目的の本かどうかを一目で確認したい。
- US-2: 利用者として、図書館に無い／予約したい場合に、その場から Amazon の該当ページへ移動して購入を検討したい。
