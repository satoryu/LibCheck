# Issue #9: ISBN手動入力 - 要件定義

## Problem Statement

バーコードスキャンが利用できない場合（バーコード破損、カメラ不具合、カバーで隠れている等）に、ユーザーがISBNを手動で入力して蔵書検索を行う手段がない。

## Requirements

### Functional Requirements

1. ISBN-13（13桁）および ISBN-10（10桁）の入力に対応する
2. リアルタイムバリデーションで入力中に有効性を表示する
3. チェックディジット検証を行い、無効なISBNを検出する
4. 数字のみ入力可能（キーボードタイプ: number）
5. 「検索する」ボタンで `/result/:isbn` へ遷移する
6. 「バーコードスキャンへ」ボタンでスキャン画面へ遷移できる
7. 無効なISBNでは検索ボタンが無効化される

### Non-Functional Requirements

1. バリデーションがリアルタイムで反映される（文字入力ごと）
2. キーボードが数字入力用に最適化されている

## Constraints

- IsbnValidator は Issue #8 で作成したものを共有する
- ルートは `/isbn-input` とする（design-guidelines 準拠）
- ハイフン入力は許容するが内部では除去して処理する

## Acceptance Criteria

1. ISBN入力フィールドが表示される
2. 有効なISBN入力時に「有効なISBNです」と表示される
3. 無効なISBN入力時にエラーメッセージが表示される
4. 空欄・桁数不足では検索ボタンが無効化される
5. 有効なISBN入力後に検索ボタンタップで検索結果画面へ遷移する
6. バーコードスキャン画面への遷移リンクが機能する
7. 全テストがパスする

## User Stories

1. ユーザーとして、バーコードが読み取れない書籍のISBNを手動入力して蔵書検索したい
2. ユーザーとして、入力中にISBNが正しいかリアルタイムで確認したい
3. ユーザーとして、手動入力からバーコードスキャンに素早く切り替えたい
