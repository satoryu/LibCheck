# 要件定義: 「別の本をスキャンする」ボタンで画面が真っ黒になるバグ修正 (#21)

## Problem Statement

検索結果ページの「別の本をスキャンする」ボタンを押すと、バーコードスキャナーに戻らず画面が真っ黒になる。

`BarcodeScannerPage`で`context.go()`を使って検索結果ページに遷移しているため、ナビゲーションスタックから`/scan`が消え、`pop()`で戻れなくなっている。

## Requirements

### Functional Requirements

- FR-1: 検索結果ページの「別の本をスキャンする」ボタンを押すと、バーコードスキャナーページに正しく戻ること

### Non-Functional Requirements

- NFR-1: 既存のテストが引き続きパスすること
- NFR-2: 修正に対するテストを追加すること

## Constraints

- GoRouterのナビゲーションパターンに準拠すること

## Acceptance Criteria

- AC-1: バーコードスキャン→検索結果→「別の本をスキャンする」ボタン押下でバーコードスキャナーに戻る
- AC-2: 画面が真っ黒にならない
- AC-3: すべての既存テストがパスする

## User Stories

- US-1: ユーザーとして、検索結果を確認した後に別の本のバーコードを続けてスキャンしたい。
