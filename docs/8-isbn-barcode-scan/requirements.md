# Issue #8: ISBNバーコードスキャン - 要件定義

## Problem Statement

ユーザーが書籍のISBNバーコードをカメラで素早く読み取り、蔵書検索を行う手段がない。物理的な書籍を手に取った際にバーコードをスキャンするだけで蔵書状況を確認できる機能が必要。

## Requirements

### Functional Requirements

1. カメラを起動し、リアルタイムでバーコードを読み取る
2. EAN-13形式のISBNバーコード（978/979 プレフィックス）を認識する
3. スキャンガイド枠を表示して読み取り位置を案内する
4. フラッシュライトの切替が可能
5. バーコード読み取り成功時に `/result/:isbn` へ自動遷移する
6. 読み取り成功時にハプティックフィードバック（振動）を提供する
7. 「ISBN を手動入力する」ボタンで手動入力画面へ遷移できる

### Non-Functional Requirements

1. カメラプレビューが滑らかに表示される
2. バーコードの認識が高速である（1秒以内）
3. カメラ権限の要求が適切に処理される

## Constraints

- Android のみ対応
- mobile_scanner パッケージを使用する
- カメラ権限はランタイムで要求する
- ISBN-13（EAN-13 の 978/979 プレフィックス）のみ対象

## Acceptance Criteria

1. スキャン画面でカメラプレビューが表示される
2. EAN-13バーコードを正しく読み取れる
3. ISBN以外のバーコードは無視される
4. 読み取り成功時に検索結果画面へ遷移する
5. フラッシュライトの ON/OFF が切り替えられる
6. カメラ権限が拒否された場合にエラー画面と設定誘導が表示される
7. 手動入力画面への遷移リンクが機能する
8. 全テストがパスする

## User Stories

1. ユーザーとして、書籍のバーコードをカメラでスキャンして蔵書を検索したい
2. ユーザーとして、暗い場所でもフラッシュライトを使ってスキャンしたい
3. ユーザーとして、バーコードが読み取れない場合に手動入力に切り替えたい
4. ユーザーとして、カメラ権限について適切に案内されたい
